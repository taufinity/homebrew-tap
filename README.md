# taufinity/homebrew-tap

Homebrew tap for the [Taufinity CLI](https://github.com/taufinity/cli).

## Install

```bash
brew install --cask taufinity/tap/taufinity
```

## Update

```bash
brew upgrade --cask taufinity/tap/taufinity
# or let the CLI handle it: taufinity update
```

## Uninstall

```bash
brew uninstall --cask taufinity/tap/taufinity
```
